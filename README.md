# EN2533 – Robot Design and Competition 2025

## Overview

As my first complete robotics project, this robot was designed from the ground up to autonomously navigate a multi-stage competition arena, perform object manipulation tasks, interpret environmental information, and complete a series of engineering challenges without human intervention.

The project required the integration of mechanical design, embedded systems, sensor fusion, control systems, and autonomous decision-making into a single robotic platform capable of completing all competition tasks reliably and efficiently.



## Competition Tasks

The robot was designed to complete the following six autonomous challenges:

### 1. Grid Navigation and Box Pick-and-Place
- Navigate a structured grid environment.
- Identify target locations.
- Pick up and transport boxes.
- Place objects accurately at designated positions.

### 2. Ramp Alignment, Ascending and Descending
- Set up the ramp.
- Maintain stability while climbing.
- Perform controlled descent without losing localization.

### 3. Barcode Reading
- Detect and scan barcodes placed on the arena floor.
- Decode barcode information.
- Use the extracted information for next task execution.

### 4. Circular Wall Following and Ball Collection
- Follow a circular wall using distance sensing.
- Maintain a consistent offset from the wall.
- Locate and collect balls while following the inner circular wall

### 5. Dashed Line Following and Pendulum Crossing
- Detect and follow discontinuous line markings.
- Handle temporary loss of line information.
- Safely navigate through a moving pendulum obstacle.

### 6. Ball Shooting Mechanism
- Position accurately at the target zone.
- Aim and launch collected balls toward designated targets.
